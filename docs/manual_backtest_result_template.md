# Manual Backtest Result Template

Use this template when reporting an MT5 Strategy Tester run manually. One run
should use one completed template.

```text
Strategy name:
Symbol:
Timeframe:
Date range:
Spread/model:
Starting balance:
Parameters/settings file:
Total net profit:
Profit factor:
Expected payoff:
Max balance drawdown:
Max equity drawdown:
Relative drawdown:
Total trades:
Win rate:
Average win:
Average loss:
Largest win:
Largest loss:
Consecutive losses:
Recovery factor:
Sharpe/statistics if available:
Notes about bad behavior:
Screenshot/report available? yes/no
User verdict:
```

Treat manually reported results as research notes. They are useful for fast
iteration, but they are not formal evidence, profitability proof, or approval
for Trial, Surge 2 Step, Vanguard, Challenge, Verification, Funded, or live-money
execution.
