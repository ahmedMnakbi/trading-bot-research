# Human Review Package

The human review package includes release metadata, the feature matrix, safety audit summary, known limitations, command reference, final check result, and a human review report.

Reviewers should inspect safety metadata, forbidden-feature statements, release verification output, audit status, limitations, and command coverage.

Disqualify the release if any artifact claims live trading, real orders, private API usage, missing safety docs, or unsafe metadata.

Human review does not approve live trading. It only reviews the non-live research and paper-trading release.
