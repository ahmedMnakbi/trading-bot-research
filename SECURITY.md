# Security

No API keys are needed for non-live mode.

Never commit `.env`.

Never paste real secrets into config files.

Logs and artifacts should be checked before sharing.

Safety audit and archive tools exclude secret-like files.

Live/API-key support is not implemented.

MT5 account trading credentials, balance fetching, position management, and live account execution are not implemented. The MT5 roadmap is documentation/config only until later gated phases are explicitly approved.
